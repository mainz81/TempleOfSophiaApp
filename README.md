# Temple of Sophia v0.1

A private writing sanctuary for book creation, Sophia-style prose development, hierarchical manuscript planning, archive preservation, and advanced AI-assisted writing.

## Setup

```bash
cd TempleOfSophia_v0_1
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
streamlit run app.py
```

Edit `.env` and add your OpenAI API key.

## Features

* Project dashboard
* Book metadata editor
* Numerical Table of Contents builder
* Clean Writing Chamber
* Advanced Production Chamber
* Sophia Writing Modes
* Archive and Vault
* Manuscript Compiler
* DOCX Export
